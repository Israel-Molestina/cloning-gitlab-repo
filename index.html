How to clone the Repo from GitLab (Mac)

In the tutorial we will walk through the steps to clone the class repository from GitLab

Step 1: Setting up your SSH key

  1. First check to see if you have SSH keys
    • In Terminal type: ls -al ~/.ssh
    • If you see a file called id_rsa.pub you're good!
  2. Next copy your SSH key
    • In Terminal type: pbcopy < ~/.ssh/id_rsa.pub
    • This copys your SSH key
  3. Then log into your class Repo
    • https://upenn.bootcampcontent.com/upenn-bootcamp/upenn-phi-fsf-pt-12-2020-u-c
    • Go to settings, then SSH Keys
  4. Add your SSH key to GitLab
    • Paste your key into the SSH feild, give it a name then click Add Key

Step 2: Clone the Repo

  1. Go to your Class Repo and hit the blue Clone button
  2. Select SSH
  3. Go to your Terminal and type: git clone
    • Paste in the SSH link you just copied and press the return button
