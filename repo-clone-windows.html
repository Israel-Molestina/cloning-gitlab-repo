Adding SSH key to Gitlab

  1. Check if SSH key set on your machine
    2.  
    • Open Git-Bash type: ls –al ~/.ssh
      •    
    • If you see id_rsa and is_rsa.pub you have your ssh set
      •  
  2. Copy your SSH key
    3.  
    • In Git-Bash type: " clip < ~/.ssh/id_rsa.pub "
      •    
    • You shouldn’t see any kind of message when you run this command. If you do, make sure you entered it correctly.
      •  
    • Note: Do not copy anything else to your clipboard until all steps are completed. Otherwise, you’ll need to enter the copy command again.
      •  
    • Login into your Gitlab Account : https://upenn.bootcampcontent.com/upenn-bootcamp/upenn-phi-fsf-pt-12-2020-u-c
      •  
    • Navigate to your profile icon and click on Setting
      •    
    • Paste ssh key and Add Title
      •  

    4. Clone the Repo from Gitlab

    1. In Gitlab go to project detail page
      2.    
    2. Click on "clone" and copy "Clone with SSH" url
      3.    
    3. In Git-Bash type : git clone (paste the url)
